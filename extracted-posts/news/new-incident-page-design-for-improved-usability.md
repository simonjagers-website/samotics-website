Incidents need to be understood at a glance. That is why we have rebuilt the incident page in SAM4 – so every user sees exactly what they need, the moment they open it.

With the new layout, critical information is easier to spot, triage is faster, and collaboration runs smoother.
## 🔍 Built for clarity and speed


We have focused on the changes that matter most to daily users:


 	- **Clear status and severity** at the top of each page

 	- **Incident descriptions** are shown clearly and consistently

 	- **Incident activity timelines** now run chronologically, top to bottom.

 	- **Simplified layout** that removes unnecessary detail and helps teams understand the situation faster



![](https://samotics.com/wp-content/uploads/2025/07/image-1.png)

Together, these changes make it easier to stay focused on what matters most: resolving issues quickly and confidently.
## 💬 Easier to follow, easier to act


New traffic light changes are now logged as part of the incident timeline, so key transitions are never missed.

We have also simplified the comment system:


 	- New comments no longer start new threads

 	- Replies have been removed to keep timelines clean and easy to follow



It is a straightforward view, designed to support real-time decision-making.

![](https://samotics.com/wp-content/uploads/2025/07/image5.jpg)
## 🚀 Available now


The redesigned incident page is now live for all SAM4 users. It is already helping teams move faster and act with greater confidence.
